19 Jun 2016, 18:57

Domyślnie pierwsza kolumna wpisu historii powłoki zawiera tylko id wykonanej komendy. Nie ma jednak informacji o czasie jej wykonania. Aby taką informację dodać, należy ustawić format czasu jaki powinien być wpisywany w zmienną środowiskową HISTTIMEFORMAT: HISTTIMEFORMAT='%H:%M:%S %d-%m-%Y '
