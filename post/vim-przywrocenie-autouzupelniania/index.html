Vim przywrócenie autouzupełniania

19/06/2016
vim / pl

Po aktualizacji Vim przestało działać autouzupełnianie. Wymagane jest w takim przypadku przejście do katalogu wtyczki YouCompleteMe, oraz odpalenie kompilacji ze wsparciem dla języka C oraz Go.

~ $ cd  .vim/bundle/YouCompleteMe/
YouCompleteMe (master) $ ./install.py --clang-completer --gocode-completer
Go: odbiorca metody, przykłady
 
Macos usługi nasłuchujące
