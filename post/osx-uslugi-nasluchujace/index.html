Macos usługi nasłuchujące

19/06/2016
macos / bash / lsof / pl

W systemie Macos polecenie netstat zachowuje się inaczej niż pod Linuxem. Do wypisania usług nasłuchujących na portach tcp lokalnej maszyny można użyć polecenia lsof. Przykładowo:

sudo lsof -i tcp -nP -stcp:listen
